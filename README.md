# Todo-App

A simple and intuitive Todo application built with **Vue 3**, **Vite**, **Pinia**, and **Vue Router**. The aim of this project is to provide a clean, responsive interface for managing daily tasks — allowing users to create, organise, and track their to-dos efficiently.

## Features

- Add, complete, and delete tasks
- Persistent state management via Pinia
- Client-side routing with Vue Router
- Fast development experience powered by Vite

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) `^20.19.0` or `>=22.12.0`

### Installation

```bash
# Clone the repository
git clone https://github.com/JGH0/Todo-App.git
cd Todo-App

# Install dependencies
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
